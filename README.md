# WhyNotTry 🏃‍♀️

Aplicativo iOS que sugere atividades esportivas com base no perfil do usuário. Desenvolvido em Swift com SwiftUI.

## 📱 Sobre o projeto

O **WhyNotTry** surgiu da ideia de incentivar pessoas a experimentarem novos esportes e atividades físicas. O app analisa o perfil do usuário e sugere atividades compatíveis com seu estilo de vida, disponibilidade e preferências.

## 🛠️ Tecnologias

- **Swift** — linguagem principal
- **SwiftUI** — interface declarativa
- **Xcode** — ambiente de desenvolvimento
- **iOS 15+** — versão mínima suportada

## 📸 Funcionalidades

- Cadastro de perfil do usuário
- Sugestão personalizada de atividades esportivas
- Interface simples e intuitiva

## 🚀 Como rodar

```bash
# 1. Clone o repositório
git clone https://github.com/michellepichau/WhyNotTry.git

# 2. Abra o projeto no Xcode
open WhyNotTry.xcodeproj

# 3. Selecione um simulador iOS e rode com ⌘ + R
```

## 📚 O que aprendi

- Construção de interfaces com SwiftUI
- Navegação entre telas com NavigationStack
- Gerenciamento de estado com `@State` e `@ObservedObject`
